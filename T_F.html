Text and Formatting

How do you make text?

In HTML, any text that's not a tag and not CSS in the style tags is considered text, and in most cases it can be written to the screen no matter where it is in the HTML code. However, this is considered very bad practice, and it can make it a nightmare to use CSS and make the website look presentable. Therefore, there are tags we can use to put the text inside them on the screen, with different qualities.

The main HTML tags used to contain text are headers 1 <h1></h1> to 6 <h6></h6>, and the paragraph tag <p></p>.

What about making your text fancy?

To add extra effects to your text, you can use the <em></em> tags for italics, <strong></strong> for bold, and <u></u> tags for underline.

This can also be done in CSS: font-style: italic; for italicized text, font-weight: bold; for bolded text, and text-decoration: underline; for underlined text. Additionally, text colour can be set using the color: command, and the value can be one of the 140 predefined colours in CSS, or an RGB or a Hex value if you want to be able to better control the shade of the colour. Finally, the font size can be increased using the font-size command, which can take values in many units. Using the unit em is the most recommended, as it is a factor of the pase paragraph size, which is 1em, and therefore scales well.

Special characters can also be added in HTML. To add a special character in text, you must start with an ampersand, then enter the code specific to that character, and then end with a semicolon. For example, if you were to type a greater than or less than bracket in HTML, it would automatically be considered as the start or end of a tag, and would cause an error. Therefore, you must use the special characters: &lt; for less than and &gt; for greater than. You can add emojis to your text as well, as long as you know their specific codes, which you can find through a quick Google search. For example, ⚡ is &#9889;, and ⚓ is &#9875;.
