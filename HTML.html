HTML Basics

What is HTML?

HTML, which stands for HyperText Markup Language, is a coding language that makes up the base of every webpage on the internet. By using HTML, you can also make your own website. Here is an example of an extremely simple website:


  <!DOCTYPE html>
  <html>
    <head>
      <title>Page Title</title>
    </head>
    <body>
      <h1>This is a Heading</h1>
      <p>This is a paragraph.</p>
    </body>
  </html>

The above code may look a bit confusing to those who don't know HTML, but it's actually really simple when it's broken down. Firstly, HTML uses tags in order to control every aspect of the website, both those that are seen and those that are not. Tags are surrounded by <> brackets, and for most tags, but not all, there will be an end tag, which is signified by a / before the name inside the angle brackets. This means that the tag contains everything from the start tag to that end tag.

What are some mandatory HTML tags for making a website?

All html pages start with the <!DOCTYPE html> tag, which is an empty tag. It's called empty because it has no end tag, and therefore does not contain any text or elements. This tag lets the website know that the file is written in HTML when it reads your code to make the website. The following <html> and </html> tags must surround the entire HTML code, and everything inside it will be read as HTML code instead of text.

The <head></head> tags surround all the data that is not seen on the website itself. This includes language, tab title, and stylesheets, which will be discussed further in-depth in the later pages of this website. In this case, the <title></title> tags inside the head element are setting the text inside, Page Title, as the tab title, which will appear as the text on the tab at the top of the search engine.

The <body></body> tags, as their nams suggests, contain the body of the website, which is everything you can see. There are a multitude of HTML elements that can be added in the body section of the website, from headers, paragraphs, and other text to links, images, and videos, and so much more! In this case, the body section contains a header, (the <h1></h1> tags) which reads "This is a Heading", and a paragraph, (the <p></p> tags) which reads "This is a paragraph."

And that's it! It's as simple as that! However, if you want to make your website a bit more complex than that simple example, the following table will list some other commonly used html elements you might want to add to your repertoire:

HTML Tag Description
<!-- --> This is a unique tag, which represents a comment. It is an empty tag, but there can be anyting in between the start of the bracket, <!--, and the end of the bracket, -->. Any text in between the two is completely ignored when creating the website from the HTML file, and so this tag's only use is to leave comments explaining the code or giving other information to anyone who would read the HTML file.
<br> This is an empty tag. Its purpose is to move down a line vertically, similar to hitting the ENTER key on the keyboard when typing.
<center></center> Anything between these HTML tags will be centered horizontally on the webpage.
<form></form> With many arguments and many options for tags that should go inside it, the form tag is certainly complex. This tag creates a fillable form. There are HTML tags for different types of questions that can be added to the form, and an argument to determine how the data is kept or used.
<input> A tag that is used inside the form tag in order to add responses to the form, for visitors to the website to fill out. It has arguments for the type of response method it should be.
<hr> Creates a horizontal line across the webpage.
<img> Display an image to the screen. Uses the attribute src to get the image.
<link> Used to reference a separate code file. Not to be confused with a hyperlink, which has a tag of a. Used in the head section of an HTML code.
<meta> Controls a variety of metadata about the website, such as character set, page description, keywords, author of the document, and viewport settings. Used in the head section of an HTML code.
<ol></ol> Contains an ordered (numbered) list. The list elements go inside this tag.
<ul></ul> Contains an unordered (bulleted) list. The list elements go inside this tag.
<li></li> List elements. Each li element contains the text or any other content that is meant to be listed in the unordered or ordered list that this element is a child of.

With all of these elements, and many more, you will be able to make stunning websites!
