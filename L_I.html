Links and Images

How do you make links?

In HTML, a link is any element or elements surrounded by an <a></a> tag. The starting tag should have an href attribute, which should be set to the value of the website's URL, or if you have another HTML file on your computer, if can link to that file path instead. That's all you need!

You can put link tags around text, around images, and even around dividers, which will be covered in the next section. In fact, the navigation bar at the top of this screen uses <a></a> tags in order to link to the other webpages on this site, so that you can move between pages without any issues.

How do you make images?

To add an image to your website, you use the <img> tag. This is an empty tag, it has no end tag, and is only one line. It requires an src attribute to work, which, similarly to a link, can take an image from a url or from a file path.

You can specify the width and height attributes using the width and height commands in CSS, or by the attributes with the same name inside the tag itself. If you only specify one measurement, the other will automatically resize to keep the original ratio of the image. Also, the alt attribute can be used to display text if the image does not properly load, or for text-to-speech programs to describe the image.

How do you add a video?

You can add a video to your website using the <video></video> tags. Inside the tags, you can add <source></source> tags to add a video file, and any text inside the video tags will be used in the same way as the alt attribute is used for images. Also, adding the attribute controls without any value just allows the reader to play and pause the video, as well as adjust the volume and skip to any time they would like.
