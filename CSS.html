CSS Basics

What is CSS?

CSS, which stands for Cascading Style Sheets, is a coding language that is used alongside your HTML code in order to add, as the name suggests, style to your website. CSS controls many visual aspects of your website, including colours, fonts, sizes, and many more specific adjustments that can be used to fine-tune your website into looking exactly how you envision it to be.

How do you code in CSS?

To code in CSS, there are three steps. You must define the HTML tag or tags that you want to apply the change to, you must choose which command you want to use, and you must specify its value.

There are three ways to add CSS to your HTML website. Firstly, you can add what is called inline CSS to any empty or starting tag. This is because each HTML tag has specific attributes that can be set for it, and every tag has the style attribute, which is where you can type CSS code. For example, the following HTML tag uses inline CSS:

  <h1 style="color: red;">This is a header</h1>

For the above example, the header tag has a style attribute, which sets its colour to red. The tag the CSS applies to is only the tag with the attribute, and no others. The command used is color, which sets the colour of the element. Finally, the value of the CSS command is red, which means that the color of the specified element, in this case the header, is red.

In the other two methods, choosing a command and its value is the same:

  command: value;

However, to select the tag or tags you want to edit, you must write the tag name, followed by curly brackets {}. Anything in the curly brackets will be applied to every tag of the type listed before the brackets. For example, this is an example of CSS code;

  p {
    text-align: center;
  }

For the above example, every paragraph element will have its text alignment centered. This type of CSS command can be used inside the HTML <style></style> tags, in which case it is only applied to the HTML file it is in. Finally, you can make a .css file, where you write only in CSS, and then you can apply the CSS commands to any HTML file by using the command:

  <link rel="stylesheet" href="style.css">

where style.css can be replaced by the name of your CSS file.
